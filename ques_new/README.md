# Question Platform（问答留言交互平台）
## 项目简介
这是一个基于 **Spring Boot** 开发的轻量级在线问答留言交互平台，采用 MVC 架构 + IOC 依赖注入，无需数据库，实现了用户注册登录、消息发布、评论互动等核心功能，CSS 与 HTML 完全分离，部署简单便捷。

## 技术栈
- 后端：Spring Boot、Lombok
- 前端：HTML5、CSS3、Thymeleaf 模板引擎
- 存储：内存集合，无需数据库
- 部署：可打包为可执行 JAR 包，内置 Tomcat 容器

## 核心功能
1.  **用户模块**：账号注册（用户名唯一）、密码登录、登录态保存
2.  **消息模块**：发布问答消息、查看所有消息列表、查看消息详情
3.  **评论模块**：对消息进行评论、查看消息对应的所有评论
4.  **辅助功能**：验证码生成与刷新、页面跳转与重定向
